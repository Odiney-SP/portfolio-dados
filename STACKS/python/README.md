# Python

> Em construção contínua — cada notebook é um passo na jornada.

Python é a principal linguagem para análise de dados, automação e machine learning. Versátil, com ecossistema robusto e curva de aprendizado acessível para quem vem de outras áreas.

---

## Por que Python para dados

- Ecossistema maduro: Pandas, NumPy, Scikit-learn, Matplotlib, Plotly
- Automação de processos repetitivos e geração de relatórios
- Conexão com APIs, bancos de dados e fontes externas
- Caminho natural para machine learning e IA

---

## Roadmap de Aprendizado

```
Nível 1 — Fundamentos
└── Variáveis, tipos de dados, operadores
└── Estruturas de controle (if, for, while)
└── Funções e escopo
└── Listas, dicionários, tuplas, sets
└── Leitura e escrita de arquivos (CSV, JSON)

Nível 2 — Pandas
└── Series e DataFrames
└── Leitura de dados (read_csv, read_excel, read_sql)
└── Filtragem, seleção e indexação
└── GroupBy e agregações
└── Merge, Join e Concat
└── Tratamento de valores nulos

Nível 3 — NumPy e Estatística
└── Arrays e operações vetorizadas
└── Estatística descritiva com NumPy
└── Distribuições e amostras

Nível 4 — Visualização
└── Matplotlib: gráficos básicos e customização
└── Seaborn: visualizações estatísticas
└── Plotly: gráficos interativos

Nível 5 — Limpeza e Transformação
└── Tratamento de outliers
└── Encoding de variáveis categóricas
└── Feature engineering básico

Nível 6 — Automação
└── Scripts de ETL
└── Automação de relatórios
└── Consumo de APIs REST

Nível 7 — Machine Learning
└── Scikit-learn: regressão, classificação, clustering
└── Avaliação de modelos
└── Forecasting com statsmodels / Prophet
```
