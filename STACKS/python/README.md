# Python

Comecei Python pelos fundamentos — variáveis, loops, funções — mas o que realmente abriu o interesse foi o Pandas. Conseguir ler um CSV, filtrar linhas, agrupar por categoria e calcular médias com poucas linhas de código foi o momento em que ficou claro que Python ia ser útil no dia a dia.

Uso Python principalmente para análise de dados e automação de relatórios. Machine learning e forecasting estão na sequência.

---

## O que já uso com regularidade

**Pandas** é a principal ferramenta. Leitura de dados, filtragem, GroupBy, merge entre bases, tratamento de nulos — já flui bem. **NumPy** uso para operações numéricas quando o Pandas não é suficiente. Para visualização, estou explorando Matplotlib e Plotly.

Também uso Python para automação: scripts que leem planilhas, processam dados e geram saídas formatadas. `requests` para consumir APIs. `openpyxl` e `xlsxwriter` para gerar arquivos Excel com formatação.

## O que estou aprendendo agora

Pandas avançado está em curso — merge com múltiplas chaves, pivot\_table, resample para séries temporais. Visualização com Seaborn para análises estatísticas. Automação mais robusta com agendamento e tratamento de erros.

## Próximo passo

Machine learning com Scikit-learn — regressão e classificação primeiro, depois clustering. Forecasting com Prophet ou statsmodels para previsão de volume operacional.

`pandas` · `numpy` · `matplotlib` · `seaborn` · `plotly` · `scikit-learn` · `statsmodels` · `sqlalchemy` · `requests` · `openpyxl`
