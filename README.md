Venho estudando dados de forma sistemática há algum tempo, e este repositório é onde registro esse processo — os estudos, os projetos que estou construindo e as ferramentas que estou aprendendo a usar.

Meu interesse em dados não é genérico. Vem diretamente do trabalho: acompanhar produtividade de equipe, monitorar SLA, identificar gargalos, entender por que a meta foi ou não atingida. Esse tipo de problema é o que me fez querer aprender SQL, Python e Power BI de verdade.
