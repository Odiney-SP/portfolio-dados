## Estrutura padrão de cada projeto

```
nome-do-projeto/
├── README.md        # Contexto, objetivo, metodologia e resultados
├── dados/           # Datasets (anonimizados ou públicos)
├── notebooks/       # Análise exploratória e modelagem
├── sql/             # Queries utilizadas
├── dashboard/       # Arquivos .pbix ou prints
└── docs/            # Documentação adicional
```
