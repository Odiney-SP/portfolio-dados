# Query Library

Biblioteca de queries SQL reutilizáveis, organizadas por tipo de análise. Cada query é documentada com contexto, uso e variações.

---

## Como usar

Cada arquivo contém queries prontas para adaptar ao seu contexto. Substituir nomes de tabelas e colunas conforme necessário.
